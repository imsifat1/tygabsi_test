# TyG-ABSI Calculator

A simple calculator for BMI, TyG Index, ABSI, and TyG-ABSI from user inputs.  
This public version provides calculations only.

## Use
- Enter fasting triglycerides (mg/dL or mmol/L) and glucose (mg/dL or mmol/L).
- Enter weight (kg), height (cm), waist (cm).
- The app converts units automatically and shows:
    - BMI
    - TyG Index
    - ABSI
